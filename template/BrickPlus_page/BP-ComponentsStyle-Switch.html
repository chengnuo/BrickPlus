按钮中的加载动画

  • 给 .btn 系列的按钮元素引用 .btn-spinner 类，得到 loading 状态下的按钮样式。它拥有和 .disabled / [disabled] 一样的设计用意，并加载等待的效果。
  • 但 input 标签没有 :before，所以在 .btn 系列按钮元素中，input 的按钮不会显示 spinner 动画。
  • 在按钮中引用的 .btn-spinner 的同时，需要脚本控制给拥有 [disabled] 属性的按钮标签添加这个属性，才能真整实现加载状态下的按钮不可用。
实例
            
<button class="btn default btn-spinner" disabled>
  ...
</button>
