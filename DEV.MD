# BrickPlus

## 简介

BrickPlus 是部分基于 Bootstarp 框架的FrontUI团队的项目UI

## 快速开始

    gulp

或者

    npm run dev

### 任务列表

* `gulp default:update` 带更新BMixin
* `gulp components` js组件开发任务
* `gulp` 默认 UI 任务

## 更新文档

    gulp deploy

或者

    npm run deploy

## 发布

  npm run bundle


## 发布到 npm

  npm publish

## git tag

  打上版本标签

## bower register

  bower 注册
